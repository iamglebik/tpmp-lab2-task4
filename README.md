# tpmp-lab2-task4 - Работа с файлами (Вариант 20)

## Overview
Программа для обработки данных пенсионеров из текстового файла.

## Author
Sinyakov Gleb, группа 12а

## Usage
1. Сборка: `make`
2. Запуск: `./bin/program`
3. Очистка: `make clean`

## Additional Notes
- Входной файл: `data/pensioners.txt`
- Выходной файл: `data/pensioners_over_5_years.txt`
- Программа определяет текущий год автоматически
- Пенсионный возраст: мужчины 65 лет, женщины 60 лет
